Wanlian.AI

Privacy Policy

Effective date: April 19, 2026

This privacy policy explains how Wanlian.AI processes information when users analyze images and generate reusable AI prompts through the browser extension.

1. What Wanlian.AI Does

Wanlian.AI is a browser extension that helps users analyze images on web pages and generate reusable AI prompts in simple, detailed, and JSON formats.

2. Information Processed by the Extension

When a user actively uses the extension, Wanlian.AI may process:

  • API credentials provided by the user, such as OpenAI, Gemini, or compatible API keys
  • Image URLs, image content, page titles, alt text, page links, and related web page context needed to generate prompt results
  • Extension settings stored locally, including provider, model, language preference, base URL, and prompt template settings

3. How Information Is Used

  • To provide the extension's core image prompt analysis functionality
  • To send user-requested image analysis requests to the AI service selected by the user
  • To store local preferences and user settings inside the browser

4. Data Storage

User settings and API keys are stored locally in the browser using extension storage. Wanlian.AI does not provide its own account system or remote cloud storage for extension user data.

5. Third-Party Services

When a user chooses to analyze an image, the extension may send relevant image data and page context to third-party AI API providers configured by the user, such as OpenAI, Gemini, or compatible API services. Data handling by those providers is governed by their own privacy policies and terms.

6. Data Sharing and Selling

Wanlian.AI does not sell user data and does not use user data for advertising purposes.

7. User Control

Users control when image analysis requests are made. Users may remove stored settings or uninstall the extension at any time.
